操作步骤
  1. 1
    拿条网线把电脑和小盒子接起来，把电脑的本地 IP 地址改为 169.254.255.100。
  2. 2
    按 Win + R，输入 cmd，回车打开命令提示符。
  3. 3
    SSH 登录盒子（使用小盒子默认 IP 169.254.255.10）：
    CMD 
    ssh pi@169.254.255.10
  4. 4
    SSH 首次连接时，会提示确认主机指纹，输入 yes 回车：
    CMD 
    Are you sure you want to continue connecting (yes/no/[fingerprint])?
    💡
    输入 yes 回车即可，此提示仅首次连接时出现。
  5. 5
    输入账户密码：（鼠标右键直接粘贴，无回显属正常）
    账户密码 
    sixelp@2024#168..
  6. 6
    写入 Netplan 配置（设置固定 IP）：
    bash — netplan config 
    sudo tee /etc/netplan/00-installer-config.yaml >/dev/null <<'EOF'
network:
  version: 2
  renderer: networkd
  ethernets:
    eth0:
      dhcp4: yes
      addresses:
        - 192.168.110.50/24    # 需要修改的 IP
      nameservers:
        addresses: [114.114.114.114]
EOF
    回车确认。
    💡

    需要配置多个 IP 时，在 addresses 下增加条目即可：

    addresses:
  - 192.168.110.50/24    # 第一个 IP
  - 192.168.111.50/24    # 第二个 IP
  7. 7
    若提示输入 sudo 密码，输入账户密码后回车（不显示字符属正常）；不提示则直接执行下一步。
    bash 
    [sudo] password for pi:  # 输入密码后回车，无回显属正常
  8. 8
    应用配置，立即生效：
    bash 
    sudo netplan apply    # 立即生效，无需重启
  9. 9
    验证 IP 是否生效：
    bash 
    ip a    # 查看当前所有 IP 地址
    ✅ 完成 在输出中看到配置的 IP 地址即表示设置成功。
